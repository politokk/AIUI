# Design System

The design system package provides a comprehensive collection of reusable UI components, utilities, and design tokens to ensure consistency across your application.

## Overview

This package includes:

- **UI Components**: Pre-built, accessible components using Radix UI primitives
- **Design Tokens**: Colors, typography, spacing, and other design constants
- **Utilities**: Helper functions and CSS utilities
- **Theming**: Dark mode support and customizable themes

## Getting Started

The design system is automatically available in your Next.js applications. Simply import the components you need:

```tsx
import { Button, Card, Input } from "@repo/design-system/components/ui"

export function MyComponent() {
  return (
    <Card>
      <Input placeholder="Enter your name" />
      <Button>Submit</Button>
    </Card>
  )
}
```

## Component Categories

### Form Elements
- Buttons, inputs, selects, checkboxes, and more
- Form validation and error handling
- Accessible form patterns

### Layout & Navigation
- Cards, modals, sheets, and dialogs
- Navigation menus and breadcrumbs
- Grid and flex utilities

### Data Display
- Tables, lists, and data grids
- Charts and visualizations
- Progress indicators

### Feedback & Communication
- Alerts, toasts, and notifications
- Loading states and skeletons
- Error boundaries

## Customization

The design system can be customized through:

- **CSS Custom Properties**: Override design tokens
- **Tailwind Configuration**: Extend the color palette and utilities
- **Component Variants**: Use built-in variants or create custom ones
