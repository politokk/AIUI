## Banner

### Usage in Layout

To use the banner in your root layout for site-wide announcements:

```tsx
import { Banner } from 'fumadocs-ui/components/banner';

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body>
        <Banner id="announcement">
          ðŸš€ New documentation site launched!
        </Banner>
        {children}
      </body>
    </html>
  );
}
```

**Note:** Set `changeLayout={false}` when displaying banners inline in documentation to prevent layout modifications.

## Steps

Steps are perfect for tutorials and guides.

<Steps>
  <Step>

    ```bash
    pnpm install fumadocs-ui fumadocs-core
    ```
  </Step>
  
  <Step>
    ### Configure Your Project
    
    Create a `source.config.ts` file in your project root:
    
    ```typescript
    import { defineConfig } from 'fumadocs-core/config';
    
    export default defineConfig({
      // Your configuration
    });
    ```
  </Step>
  
  <Step>
    ### Start Building
    
    You're all set! Start creating your documentation:
    
    ```bash
    pnpm dev
    ```
  </Step>
</Steps>
